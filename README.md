# Artisan Coffee

## Overview

Artisan Coffee is a React-based web application that simulates a real-world coffee shop platform with both an admin dashboard and a customer-facing shop interface. It demonstrates full CRUD functionality connected to a live REST API, focusing on clean state management, reusable components, and data synchronization across views.

The project ensures that updates made in the admin dashboard are immediately reflected in the shop through a centralized API.

## Features

- Full CRUD functionality (Create, Read, Update, Delete)
- Admin dashboard for managing coffee products
- Customer-facing shop interface
- Search functionality for filtering products by name
- Location-based filtering system
- Real-time synchronization with REST API
- Clean and responsive UI design

---

## Tech Stack

- React (useState, useEffect)
- Fetch API
- JSON Server (hosted backend on Render)
- Tailwind CSS
- Vite

---

## Key Learning Outcomes

- API integration in React applications
- State management across multiple components
- Building CRUD functionality with REST APIs
- Handling real-time UI updates from backend changes
- Debugging data synchronization issues between frontend views

---

## Setup Instructions

- Clone the repository
- Run `npm install`
- Run `npm run dev`
- Ensure API is running or connected to deployed endpoint
